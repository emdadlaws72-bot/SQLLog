# MySQL 实时SQL监控小工具 V2.0

## 📖 产品简介

**仓储现场开发/测试/定位问题专用工具**

无需部署，无需安装依赖，开箱即用，实时监控 MySQL 所有执行的 SQL 语句。

## 🚀 快速开始

### 一键启动（推荐）

1. 双击 `启动工具.bat`
2. 等待自动打开浏览器（访问 http://localhost:3003）
3. 输入数据库连接信息（支持自动记忆）
4. 点击"连接数据库"
5. 点击"开始监控"

### 传统方式

```bash
npm install
npm start
```

## 🎯 功能特性

### 核心功能
- **实时监控**：每秒抓取 performance_schema 的 SQL 执行记录
- **自动保存**：每 2000 条 SQL 自动保存为 TXT 文件并清空
- **停止保存**：停止监控时自动保存当前数据
- **历史查看**：查看所有历史保存的 SQL 日志文件

### 筛选功能
- **SQL 类型筛选**：SELECT/INSERT/UPDATE/DELETE/全部
- **执行时间筛选**：只显示执行时间≥配置值的 SQL
- **业务类型筛选**：入库/出库/补货/盘点/空容器/库存管理
- **数据库筛选**：连接后自动获取数据库列表
- **表名筛选**：选择数据库后自动获取表名
- **涉及表筛选**：实时识别 SQL 中涉及的表

### 体验优化
- 🔒 不自动跳转到最新 SQL，可固定查看任意一条
- 📐 固定页面宽度，排版整齐美观
- 🎨 SQL 语句格式化显示，关键字高亮
- 📜 历史日志可滚动、可格式化、可筛选
- 💾 配置自动保存，下次打开自动记忆
- 🔄 文件列表自动刷新
